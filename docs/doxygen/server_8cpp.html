Detailed Description

Implementation of the SafeSocket multi-client TCP server.

Provides the full implementation for Server as declared in server.hpp. The server supports:

  • Concurrent clients via per-client detached threads (Server::client_loop).
  • An accept thread (Server::accept_loop) that enqueues new connections.
  • An optional keepalive thread (Server::keepalive_loop) that periodically pings all connected clients.
  • Broadcast and private text messaging.
  • Bidirectional file transfer (client?server upload and server?client push).
  • Access-key authentication, nickname management, and client kicking.
  • An interactive operator CLI (Server::run_cli).
