Detailed Description

Fixed-size packet header transmitted before every payload.

The header is exactly 20 bytes and is packed to avoid compiler-inserted padding. It must be sent and received as a complete atomic unit before the accompanying payload bytes.

Wire layout (20 bytes total):

Offset Size Field
0 4 magic
4 4 msg_type
8 4 sender_id
12 4 target_id
16 4 data_len
