Detailed Description

Implementation of the SafeSocket TCP client.

Provides the concrete implementations for all methods declared in client.hpp. The client runs two concurrent threads:

  • The main thread (caller of Client::run_cli()) reads commands from stdin and translates them into outgoing packets.
  • The receive thread (Client::recv_loop()) blocks on the socket and dispatches incoming packets via Client::handle_packet().
