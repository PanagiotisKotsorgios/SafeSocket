Detailed Description

Application entry point for SafeSocket.

Parses command-line arguments, loads configuration, initialises networking, and launches either the Server or the Client depending on the first positional argument (server / client). A third mode, genconfig, writes a default configuration file and exits.

Invocation Syntax

safesocket server [options] Start as a multi-client server
safesocket client [options] Connect to a server as a client
safesocket genconfig [path] Write a default .conf file and exit

Argument Precedence

Values are applied in the following order (later stages override earlier ones):

  1. Compiled-in defaults (struct member initialisers in Config).
  2. Configuration file (loaded via Config::load()).
  3. Command-line flags (--host, --port, etc.).

Application entry point.

Execution flow:

  1. On Windows, attempt to enable ANSI colour via enable_ansi_colors_windows().
  2. Print the startup banner via print_banner().
  3. Validate that a mode argument was supplied; print usage and exit if not.
  4. Handle the special genconfig mode immediately (write defaults, print config, exit).
  5. Scan argv for --config to find the config file path before the full argument parse, then call Config::load().
  6. Iterate argv a second time to apply all --flag overrides to g_config.
  7. Call net_init(); exit on failure.
  8. In server mode: warn about misconfigured security options, construct a Server, call Server::start(), then Server::run_cli().
  9. In client mode: warn about misconfigured encryption, construct a Client, call Client::connect(), then Client::run_cli().

Call net_cleanup() and return the exit code.

0 on clean exit, 1 if startup or connection failed.

Handle the genconfig subcommand.

Saves current (default) configuration to the specified path (or "safesocket.conf" if none is given), then prints the configuration and exits.

First-pass scan for the --config flag.

Scanned before the full argument loop so that the correct file is loaded before any --flag overrides are applied. Defaults to "safesocket.conf" when not specified.

Apply command-line flags to g_config.

Flags override values loaded from the configuration file. Each flag that takes a value consumes the following argv element. Unknown flags beginning with '-' are reported to stderr.

Initialise the networking subsystem.

On Windows this calls WSAStartup(); on POSIX it installs SIG_IGN for SIGPIPE. Any failure here is fatal.
