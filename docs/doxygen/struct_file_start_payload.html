Detailed Description

Fixed-length prefix of a MSG_FILE_START payload.

Immediately follows PacketHeader in a MSG_FILE_START packet. The filename string (without null terminator) is appended directly after this structure; its length is given by filename_len.

Wire layout:

Offset Size Field
0 8 file_size
8 4 filename_len
12 fname <filename bytes>
