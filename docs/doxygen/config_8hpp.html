Detailed Description

Runtime configuration storage and persistence for SafeSocket.

Declares the Config structure that holds every tunable parameter used by both the server and client, as well as the global g_config instance that is shared across all translation units.

Configuration values can be sourced from three places, in increasing priority:

  1. Compiled-in defaults (member initializers in this struct).
  2. A plain-text .conf file loaded via Config::load().
  3. Command-line arguments parsed in main.cpp.

Values can also be mutated at runtime through the /set CLI command, which ultimately calls Config::set().
