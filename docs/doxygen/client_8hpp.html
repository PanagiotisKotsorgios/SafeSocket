Detailed Description

TCP client for SafeSocket with messaging, file transfer, and CLI.

Declares the PeerInfo helper structure and the Client class. The client maintains a single persistent TCP connection to a SafeSocket server, receives incoming packets on a background thread, and exposes an interactive command-line interface that the caller drives by invoking Client::run_cli().

Typical lifecycle:

Client client;
if (client.connect("127.0.0.1", 9000)) {
client.run_cli(); // blocks until the user types /quit
}
